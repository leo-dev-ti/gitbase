Gitbase 🌙

O'que é o Gitbase?

O Gitbase é uma biblioteca node simples criada com o Octokit para a criação de banco de dados simples utilizando o Github e seus issues para seus projetos simples sem limites de dados, úsuarios conectados e nem de requests.

Vantagens 👌

  • De grátis 🤝
  • Sem limites 🤩
  • Hue Hue BR 😎
  • Código aberto 👀
  • Mais fácil que enem 📚

Desvantagens 👎

  • Não pode fazer um bolo de morando ❌
  • Não ajuda contra a calvície 🚨
  • Não tem mundial 🏆
